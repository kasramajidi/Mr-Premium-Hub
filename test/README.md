# نسخه تست استاتیک ريتكس (فقط HTML, CSS, JS)

این پوشه همان ظاهر و خروجی **raitx-platform** را با فقط سه تکنولوژی HTML، CSS و JavaScript (بدون فریمورک) شبیه‌سازی می‌کند. تمام صفحات سایت اصلی به صورت استاتیک اضافه شده‌اند.

## فونت

فونت **IRANSans** (همان فونت سایت اصلی) داخل پروژه تست قرار دارد:
- `font/woff2/IRANSansWeb.woff2`
- `font/woff2/IRANSansWeb_Bold.woff2`
- `font/woff2/IRANSansWeb_Light.woff2`
- `font/woff2/IRANSansWeb_Medium.woff2`

این فایل‌ها از `raitx-platform/public/font/woff2/` کپی شده‌اند.

## نحوه اجرا و دیدن خروجی

1. **کپی کردن تصاویر از پروژه اصلی (اختیاری برای لوگو و صفحه درباره‌ما)**
   - از مسیر `../raitx-platform/public/Images` پوشه `Images` را کپی کنید داخل `test/` (نتیجه: `test/Images/`) تا لوگو، نماد اعتماد و پس‌زمینه بخش آمار صفحه درباره‌ما (`Images/bg-img2.png`) درست نمایش داده شوند.
   - برای عکس هیرو صفحه درباره‌ما (عکس تیم): فایل `piic2.png` را از `../raitx-platform/public/` کپی کنید داخل `test/` (نتیجه: `test/piic2.png`).
   - اگر این فایل‌ها را کپی نکنید، در صفحه درباره‌ما به‌جای آن‌ها تصویر جایگزین نمایش داده می‌شود.

2. **چطور خروجی (سایت) را ببینم؟**
   - **ساده:** روی `index.html` دابل‌کلیک کنید تا صفحهٔ اصلی در مرورگر باز شود.
   - **پنل ادمین:** روی `admin/index.html` دابل‌کلیک کنید (یا بعد از باز کردن سایت به مسیر `admin/index.html` بروید). در صفحه لاگین هر نام کاربری/رمز وارد کنید و «ورود به پنل» را بزنید.
   - **با سرور محلی (برای مسیرها و درخواست‌ها):**
     - در ترمینال داخل پوشه `test`: `npx serve .`
     - یا در Cursor/VS Code: راست‌کلیک روی `index.html` → **Open with Live Server**
   - آدرس API پنل ادمین همان `https://mrpremiumhub.org/api.ashx` است (مطابق raitx-platform).

## ساختار و صفحات

- `index.html` — صفحه اصلی (هیرو، خدمات، پرداخت، آزمون، اکانت پریمیوم)
- `about.html` — درباره ما (هیرو، آمار، نظرات، مقالات، تماس)
- `contact.html` — ارتباط با ما (هدر، کارت‌های ویژگی، فرم تماس، نقشه)
- `faq.html` — سوالات متداول (سایدبار دسته‌ها، آکوردیون سوال/جواب)
- `news.html` — اخبار و مقالات (صفحه لیست)
- `auth.html` — ورود و ثبت نام (فرم ورود)
- `valid-cards.html` — کارت های اعتباری
- `currency-payment.html` — پرداخت ارزی
- `cart.html` — سبد خرید
- `checkout.html` — تکمیل خرید
- `checkout-success.html` — موفقیت پرداخت
- `my-account.html` — حساب من

- **پنل ادمین** (`admin/`): همان APIهای raitx-platform (`mrpremiumhub.org/api.ashx`) — داشبورد، مقالات، محصولات، سفارشات، کاربران، انتقادات و پیشنهادات (ExamRegister)، کامنت فروشگاه، کامنت مقاله‌ها، پشتیبانی، تنظیمات.
- `css/styles.css` — استایل‌های یکسان با سایت اصلی (فونت، رنگ برند، هدر، فوتر، صفحات درباره/تماس/FAQ)
- `css/admin.css` — استایل پنل ادمین (سایدبار، کارت‌ها، جدول، فرم، مودال)
- `js/app.js` — منوی موبایل، دراپ‌داون خدمات
- `js/api.js` — آدرس پایه API
- `js/admin-layout.js` — سایدبار و چک لاگین ادمین
- `font/woff2/` — فونت IRANSans (کپی شده از raitx-platform)
- `Images/` — بعد از کپی از `public/Images` (لوگو، نماد اعتماد، تصاویر)
